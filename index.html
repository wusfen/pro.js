介绍

proto.js 是一个js增强库，提供许多实用的链式方法，使数据操作更加方便。 同时还提供了许多 polyfill ，如 array.forEach, array.map 等。

日期操作

today, addFullYear, addMonth, addDate, format, ...

                    
Date.today().addDate(-3).format('yyyy-MM-dd')

数组操作，CRUD

增insert, 查select,get, 改update, 删delete, 分页page, 排序orderBy, 分组groupBy, 去重uniq, 乱序shuffle, ...

并集union, 交集same, 异或集xor, 差集difference, 计算: min, max, avg

                    
var list = [
    {id:1, name:'apple', country:'中国'},
    {id:2, name:'banana', country:'美国'},
    {id:3, name:'wsf', country:'中国'},
]

list.select({country:'中国'})
                    
                
                    
list.get({id:1}).name

Array.prototype.select(where)

where: 查询条件

1. where 默认匹配规则： 基本类型转为String后比较，如 1 和 '1' ，true 和 'true' ， NaN 和 NaN 等 视为相等，复杂类型则递归按上述规则对比

                    
var list = [
    {id:1, name:'wsf'},
    {id:2, name:'wsf2'},
    {id:3, name:'wsf'},
]

list.select({name:'wsf'})

2. where 的key带上比较符： '==' '===' '!=' '!==' '>' '>=' '<' '<='

                    
var list = [
    {id:1, name:'wsf', age:10},
    {id:2, name:'wsf2', age:17},
    {id:3, name:'wsf', age:18},
]

list.select({'age>=': 18})

3. where 使用正则进行匹配

                    
var list = [
    {id:1, name:'wsf'},
    {id:2, name:'wsf2'},
    {id:3, name:'wsf'},
]

list.select({name: /f2/})

* 其它使用 where参数 的方法，皆与 select 匹配规则一致
